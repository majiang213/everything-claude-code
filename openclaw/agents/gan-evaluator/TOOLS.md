# TOOLS.md - Available Tools

## Core Tools

| Tool | Description |
|------|-------------|
| Read | 读取文件内容 |
| Write | 创建或覆盖文件 |
| Edit | 精确编辑文件 |
| Glob | 文件名匹配搜索 |
| Grep | 内容搜索 |
| Bash | Shell 命令执行 |

## Agent Tools

- `Read`
- `Write`
- `Bash`
- `Grep`
- `Glob`

## Usage

通过 /dispatch 命令或直接调用触发。
